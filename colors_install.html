Installing my custom colorscheme for Dwarf Fortress.

Installing custom color schemes and tilesets for Dwarf Fortress is easy. This is the same process for both the "classic" free version of the game from the Bay 12 Games website and the "premium" paid-for version of the game.

I would create backups of the files I'm going to ask you to modify before actually modifying them so you can easily switch back to the game's defaults later if you want to.

Installing the tileset

* Download this image.

* Save or move the image to the `data/art/` folder in your Dwarf Fortress directory.

* Navigate to `data/init/` in your Dwarf Fortress directory and open init_default.txt in a text editor. Change the entries after `FONT:`, `FULLFONT:`, and `BASIC_FONT` to kmkw_2x.png as shown in the image below.

Installing the colors

* Download my color scheme from here. You can just click the "Download raw file" button on the far right to download the file directly.

* Navigate to the `data/init/` folder in your Dwarf Fortress directory. Drop my colors.txt file into the folder and replace the file.

That's it, enjoy your new eye-saving tiles and colors.
